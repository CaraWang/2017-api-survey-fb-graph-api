    Facebook Graph API

  • 物件 之間的 關聯，會形成 Graph
  • 物件：獨立可以存在的東西
  • 例如：人、粉絲專頁、社團、活動
  • 關聯：依賴於某個物件所存在的東西，透過這個東西可以找到其他物件
  • 例如：貼文、照片、讚、留言
